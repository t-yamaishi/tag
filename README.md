# README


* 開発言語
ruby 3.0.1
rails 6.1.7
* 就業タームの技術
device
AWS EC2
* カリキュラム外の技術
ransack
TwitterAPI
* 実行手順
```
$ git clone git@github.com:t-yamaishi/to_do_app.git
$ cd to_do_app
$ bundle install
$ rails db:create && $ rails db:migrate
$ rails s
```
